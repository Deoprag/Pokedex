# Pokedex
Pokedex feita em HTML, CSS e JS com a utilização da API Pokeapi

## Funcionalidades:
- Pesquisa pelo nome do Pokemon
- Pesquisa pelo número do Pokemon
- Ver os pokemons que vem antes ou depois do pokemon atual
